Howto:HTML で動画を埋め込んで再生する

bereful8797著

HTML に埋め込んだ動画を再生します。動画ファイルは、ogv形式で用意することをおすすめします。これは自由ソフトウェアフォーマットで、著作権や特許の問題がほとんどありません。多くのデバイスが再生をサポートしています。

方法

以下のコードをHTMLに貼り付けてください。動画ファイル(ogv)は、HTMLと同じフォルダに置いてください。「縦幅」「横幅」には、動画のサイズをピクセルで入力してください。「ファイル名」はあなたの動画のファイル名で置き換えてください。

<video width="横幅" height="縦幅" controls>
<source src="ファイル名.ogv" type="video/ogg">
残念ながら、あなたのブラウザでは動画を再生できません。ブラウザをアップデートしてください。
</video>

ogv のエンコードについて

ffmpeg を使用すると、容易に ogv 形式の動画を用意できるでしょう。これは自由ソフトウェアです。GNU/Linuxでは次のコマンドを使用してください。冒頭の ~$ を入力する必要はないでしょう。「元動画のパス」にはファイルの場所を入力してください。よくわからなければ、-i を入力したあとに1回スペースキーを押し、ファイルをドラッグアンドドロップしてください。

動画の画質が悪いと感じられる場合は、"85x480"を"1280x720"にしたり、"3000k"を"8000k"にしてみてください。ただしファイルサイズは大きくなり、あなたのウェブサーバへの負荷も大きくなるでしょう。

~$ ffmpeg -i 元動画のパス -r 30 -s 853x480 -b:v 3000k 出力ファイル名.ogv

デモ

このビデオを再生すると、およそ6.1MiBのデータ通信量が使用されます。

github の容量制限により、このムービーは冒頭の30秒のみを切り出しています。このムービーのフルバージョンをご覧になりたい場合は、PeerTubeでご覧いただけます。
